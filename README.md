# Flibusta Telegram Downloader Bot

Telegram-бот для поиска и загрузки книг из библиотеки Flibusta через OPDS API.

## Описание

Бот позволяет пользователям искать книги в библиотеке Flibusta и загружать их в различных форматах (FB2, EPUB, MOBI, PDF, TXT, RTF, HTML) прямо в Telegram. Поиск осуществляется через OPDS API, результаты отображаются с пагинацией, а информация о книгах кэшируется для быстрого доступа.

## Основные возможности

- **Поиск книг** - поиск по названию или автору через OPDS API Flibusta
- **Просмотр результатов** - отображение результатов поиска с пагинацией (по 5 книг на странице)
- **Информация о книге** - просмотр детальной информации о книге с обложкой
- **Загрузка книг** - скачивание книг в различных форматах (FB2, EPUB, MOBI, PDF, TXT, RTF, HTML)
- **Кэширование** - кэширование результатов поиска и информации о книгах для быстрого доступа
- **Прямые ссылки** - возможность открыть книгу на сайте Flibusta

## Технологии

- **Python 3.x**
- **aiogram** - фреймворк для создания Telegram-ботов
- **aiohttp** - асинхронный HTTP-клиент
- **python-dotenv** - загрузка переменных окружения

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/overklassniy/flibusta_tg_downloader.git
cd flibusta_tg_downloader
```

2. Создайте виртуальное окружение:
```bash
python -m venv .venv
```

3. Активируйте виртуальное окружение:
- Windows:
```bash
.venv\Scripts\activate
```
- Linux/Mac:
```bash
source .venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Создайте файл `.env` на основе `env.example`:
```bash
cp env.example .env
```

6. Отредактируйте `.env` и укажите токен вашего Telegram-бота:
```
BOT_TOKEN=your_bot_token_here
```

Для получения токена бота:
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Следуйте инструкциям для создания бота
3. Скопируйте полученный токен в файл `.env`

### Настройка белого списка пользователей (опционально)

Если вы хотите ограничить доступ к боту только определенными пользователями, добавьте в файл `.env`:

```
WHITELIST_USER_IDS=123456789,987654321
```

Где `123456789` и `987654321` - это ID пользователей Telegram (можно узнать через бота [@userinfobot](https://t.me/userinfobot)).

**Важно:**
- Если переменная `WHITELIST_USER_IDS` не указана или пуста, доступ к боту открыт для всех пользователей
- Если переменная указана, только пользователи из списка смогут использовать бота
- Пользователи, не входящие в белый список, не получат никаких ответов от бота

## Запуск

Запустите бота:
```bash
python bot.py
```

Бот будет готов к работе и начнет обрабатывать команды и сообщения от пользователей.

## Использование

### Команды бота

- `/start` - приветственное сообщение и инструкции
- `/help` - справка по использованию бота
- `/search <запрос>` - поиск книг по запросу

### Поиск книг

Просто отправьте боту название книги или имя автора в текстовом сообщении. Например:
- "Война и мир"
- "Толстой"
- "Гарри Поттер"
- "Достоевский Преступление"

### Работа с результатами

1. После поиска вы увидите список найденных книг с пагинацией
2. Нажмите на кнопку с названием книги для просмотра детальной информации
3. Выберите формат для загрузки (FB2, EPUB, MOBI, PDF и т.д.)
4. Книга будет загружена и отправлена вам в Telegram

## Конфигурация

Основные настройки находятся в файле `config.py`:

- `BOT_TOKEN` - токен Telegram-бота (из переменной окружения)
- `WHITELIST_USER_IDS` - белый список ID пользователей (из переменной окружения, опционально)
  - Если не указан или пуст, доступ открыт для всех
  - Если указан, только пользователи из списка имеют доступ
- `FLIBUSTA_OPDS_BASE_URL` - базовый URL OPDS API Flibusta
- `FLIBUSTA_BASE_URL` - базовый URL сайта Flibusta
- `MAX_SEARCH_RESULTS` - максимальное количество результатов поиска (по умолчанию 20)
- `RESULTS_PER_PAGE` - количество результатов на странице (по умолчанию 5)
- `MAX_FILE_SIZE` - максимальный размер файла для загрузки (по умолчанию 50 МБ)
- `REQUEST_TIMEOUT` - таймаут HTTP-запросов в секундах (по умолчанию 30)

## Особенности реализации

- **Асинхронная архитектура** - все операции выполняются асинхронно для лучшей производительности
- **Кэширование** - результаты поиска и информация о книгах кэшируются для уменьшения количества запросов к API
- **Обработка ошибок** - корректная обработка ошибок сети, парсинга и загрузки файлов
- **Пагинация** - результаты поиска разбиты на страницы для удобной навигации
- **Поддержка форматов** - автоматическое определение доступных форматов для каждой книги
- **Обработка ZIP-архивов** - автоматическая распаковка ZIP-архивов для некоторых форматов
- **Белый список пользователей** - возможность ограничить доступ к боту только определенными пользователями

## Логирование

Бот использует встроенный модуль логирования Python. Логи включают:
- Информацию о запуске и остановке бота
- Запросы пользователей
- Результаты поиска
- Ошибки и предупреждения

## Ограничения

- Максимальный размер файла для загрузки: 50 МБ (настраивается в `config.py`)
- Некоторые форматы могут требовать авторизации на Flibusta и недоступны для прямой загрузки
- Результаты поиска ограничены 20 книгами (настраивается в `config.py`)
